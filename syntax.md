#### import 語法
```go
package main

import "fmt"
import "math"

import (
    "fmt"
    "math"
    )

func main() {
    fmt.Printf
}

```

#### 大寫字母開頭為export 
如果要取用package裡面的函數，必須要用大寫
e.g. math.pi => math.Pi
